# React Todo Basic

A simple React application to practice state management, event handling, and localStorage persistence.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persist tasks using localStorage

## Built With
- React
- JavaScript
- CSS

## What I Learned
- Managing arrays in React state
- Using useEffect for side effects
- Persisting data in localStorage
- Structuring a small React application

## Run Locally
npm install
npm run dev
